# even?メソッド
対象の数値が「偶数」かどうかを判断するメソッド。<br>
対象の要素の値が偶数であれば真を返し、そうでない場合は偽を返します。<br>
これはRubyが元々用意しているメソッドです。<br>

下記URLはeven?メソッドの公式リファレンスです。<br>
https://docs.ruby-lang.org/ja/latest/method/Integer/i/even=3f.html

例1
```ruby
puts 10.even?
```
```ruby
true
# ターミナルの返し
```
例2
```ruby
puts 5.even?
```
```ruby
false
 # ターミナルの返し
```
